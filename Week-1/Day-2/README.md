Week 1 Day 2 - SQL Queries Practice
Overview
This practice session covers important SQL concepts used for retrieving, filtering, grouping, and analyzing data from database tables.
The queries are written using the Employees table and include different SQL clauses and operators commonly used in real-world database operations.

Topics Covered
1. SELECT Queries
Used to retrieve data from a table by selecting specific columns or complete records.

2. WHERE Clause Queries
Used to filter records based on specific conditions.

3. GROUP BY Queries
Used to group rows having similar values and perform aggregate operations.

4. HAVING Clause Queries
Used to filter grouped data after applying aggregate functions.

5. TOP / LIMIT Queries
Used to display a limited number of records from a table.

6. DISTINCT Queries
Used to remove duplicate values and display unique records.

7. Comparison Operator Queries
Used to compare values using operators like >, <, >=, <=, and <>.

8. Logical Operator Queries
Used to combine multiple conditions using AND, OR, and NOT.

9. IN and NOT IN Queries
Used to match values from a specified list.

10. BETWEEN Queries
Used to filter values within a specified range.

11. LIKE Operator Queries
Used for pattern matching with wildcard characters.
